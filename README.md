# Phase 0 Template Files

This package contains the missing Phase 0 files for `ai-assisted-devsecops-learning-system`.

## Files

- `MANIFESTO.md`
- `templates/DECISION_LOG_TEMPLATE.md`
- `templates/POSTMORTEM_TEMPLATE.md`
- `templates/SECURITY_REVIEW_TEMPLATE.md`
- `templates/PROJECT_README_TEMPLATE.md`

## Suggested usage

Copy these files into the root of your repository, preserving the `templates/` directory structure.

```bash
cp MANIFESTO.md /path/to/ai-assisted-devsecops-learning-system/
cp -r templates /path/to/ai-assisted-devsecops-learning-system/
```

Then commit:

```bash
git add .
git commit -m "Add manifesto and reusable project templates"
git push
```
